Roadmap: GitHub Actions com Node.js

Aprenda o básico de CI/CD criando um workflow simples para projetos Node.js.

Roadmap de Estudos

Etapa 1

Entender CI/CD

Aprenda o conceito de integração contínua e entrega contínua.

Etapa 2

Conhecer GitHub Actions

Entenda o que são workflows, jobs, steps e actions.

Etapa 3

Criar a pasta correta

Todo workflow deve ficar dentro de .github/workflows.

Etapa 4

Configurar Node.js

Use a action actions/setup-node para definir a versão do Node.

Etapa 5

Rodar comandos npm

Execute comandos como npm install, npm test e npm run build.

Etapa 6

Analisar resultados

Veja os logs na aba Actions do GitHub e corrija possíveis erros.
